## Important Notes

- As of 2023, Japan is gradually opening online visa options — check embassy website for current status
- Required documents: application form, photo, passport, return ticket, hotel booking, bank statements (last 3 months), employment letter/leave letter
- Japan multiple-entry visa available for frequent travellers with previous Japan visit
- Cherry blossom season (March–April) is peak — start visa process 6–8 weeks ahead
- SkyNest Japan visa document checklist available in the app
